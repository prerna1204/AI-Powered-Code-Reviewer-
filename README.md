# AI-Powered Code Reviewer

A simple web app that reviews code using AI and helps developers improve their programs. It checks for possible bugs, readability issues, and suggests better coding practices.

This project is built using React, Tailwind CSS, and the Gemini API to provide fast and clear feedback in a clean interface.
